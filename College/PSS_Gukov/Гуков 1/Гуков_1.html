Hyper Text Markup Language (HTML) является стандартным языком, предназначенным для создания гипертекстовых документов. Использование HTML позволяет форматировать документы для их представления с использованием шрифтов, линий и других графических элементов с помощью любого ПО, поддерживающего HTML.
Начальная страница представляет собой либо документ Office, либо HTML-страницу, с которой начинается работа в Internet. Это может быть созданный электронный документ, основная страница организации, страница Всемирной паутины, расположенная в Internet и т.д. Это любая страница, которая обычно содержит большое количество гиперссылок на часто используемые документы.
Первый тэг < !DOCTYPE > со всеми его параметрами – это идентификатор, который сообщает браузеру, какая именно версия HTML была использована для создания данной страницы. С указанного адреса браузер (в случае необходимости) может загрузить описание элементов данной версии языка. Часть, касающаяся адреса, может быть опущена. Аббревиатура DTD (Document Type Definition) означает «определение типа документа».
